API-Dokumentation

Die OpenAPI-/Swagger-Dokumentation ist in den Server eingebaut.

Der Service verwendet die OpenAPI-Spezifikjation zur Dokumentation des Application Program Interfaces (API).

Nachdem der Dienst gestartet wurde, ist die Schnittstellendokumentation unter http://localhost:8080/api verfübar. Hostname, Schema und Port müssen entsprechend geändert werden, je nachdem, wo der Service läuft.

Es ist auch möglich, individuelle Endpunkte im Browser auszuprobieren.
