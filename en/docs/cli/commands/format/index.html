The Command "format"

You can get a list of all supported e-invoice output formats of E-Invoice-EU or information about a particular output format with the command format.

List Formats

You can get a list of the supported formats with the option --list.

e-invoice-eu format --list

It prints a list of format identifiers on standard output.

Information About a Format

To receive extended information about a particular format, use the option --info with the name of the format as an argument (case does not matter):

e-invoice-eu format --info=Factur-X-Extended:

The output looks like this:

name: Factur-X-Extended
syntax: CII
mimeType: application/pdf
customizationID: urn:cen.eu:en16931:2017#conformant#urn:factur-x.eu:1p0:extended
profileID: urn:fdc:peppol.eu:2017:poacc:billing:01:1.0
