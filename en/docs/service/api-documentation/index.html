API Documentation

The OpenAPI (Swagger) documentation is built into the server.

The service uses the OpenAPI specification to document its API.

After starting the service, the API documentation is available at http://localhost:8080/api. Hostname, scheme, and port have to be changed accordingly, depending where you have deployed the service.

It is also possible to try out the individual endpoints in the browser.
